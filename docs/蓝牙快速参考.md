# 蓝牙控制系统快速参考

## 快速启动

### 方法 1：使用启动脚本（推荐）

```bash
cd /home/pi/PipeBot_ws
source install/setup.bash
./src/PipeBot_bringup/scripts/start_bluetooth.sh
```

### 方法 2：使用 ROS2 启动文件

```bash
cd /home/pi/PipeBot_ws
source install/setup.bash
ros2 launch PipeBot_bringup bluetooth_control.launch.py
```

### 方法 3：手动启动各个节点

需要打开 7 个终端，分别运行：

```bash
# 终端 1
ros2 run PipeBot_driver bluetooth_receiver_node

# 终端 2
ros2 run PipeBot_driver state_manager_node

# 终端 3
ros2 run PipeBot_driver command_handler_node

# 终端 4
ros2 run PipeBot_driver bluetooth_sender_node

# 终端 5
ros2 topic echo bluetooth/status

# 终端 6
ros2 topic echo bluetooth/command

# 终端 7
ros2 topic echo robot/state
```

## 常用命令

### 检查环境

```bash
./src/PipeBot_bringup/scripts/start_bluetooth.sh --check
```

### 显示帮助

```bash
./src/PipeBot_bringup/scripts/start_bluetooth.sh --help
```

### 发送测试命令

```bash
# 前进
ros2 topic pub --once bluetooth/command std_msgs/msg/UInt8MultiArray "data: [32, 100]"

# 后退
ros2 topic pub --once bluetooth/command std_msgs/msg/UInt8MultiArray "data: [33, 100]"

# 左转
ros2 topic pub --once bluetooth/command std_msgs/msg/UInt8MultiArray "data: [36]"

# 右转
ros2 topic pub --once bluetooth/command std_msgs/msg/UInt8MultiArray "data: [37]"

# 制动
ros2 topic pub --once bluetooth/command std_msgs/msg/UInt8MultiArray "data: [34]"

# 停止
ros2 topic pub --once bluetooth/command std_msgs/msg/UInt8MultiArray "data: [35]"

# 自动模式
ros2 topic pub --once bluetooth/command std_msgs/msg/UInt8MultiArray "data: [16]"

# 手动模式
ros2 topic pub --once bluetooth/command std_msgs/msg/UInt8MultiArray "data: [17]"
```

### 监听话题

```bash
# 监听蓝牙状态
ros2 topic echo bluetooth/status

# 监听接收到的命令
ros2 topic echo bluetooth/command

# 监听机器人状态
ros2 topic echo robot/state

# 监听速度命令
ros2 topic echo cmd_vel

# 监听模式状态
ros2 topic echo robot/mode
```

### 查看节点信息

```bash
# 列出所有运行的节点
ros2 node list

# 列出所有话题
ros2 topic list

# 查看话题详细信息
ros2 topic info bluetooth/command

# 查看话题消息类型
ros2 topic type bluetooth/command

# 查看节点信息
ros2 node info bluetooth_receiver_node
```

## 配置参数

### bluetooth_receiver_node

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `bt_name` | `PipeBot Control` | 蓝牙设备名称 |
| `bt_adapter` | `hci0` | 蓝牙适配器 |

### state_manager_node

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `publish_rate` | `2.0` | 状态发布频率（Hz） |
| `distance_warning_threshold` | `200.0` | 测距警告阈值（mm） |

### command_handler_node

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `command_timeout` | `0.5` | 命令超时时间（秒） |
| `default_speed` | `100` | 默认速度（0-255） |
| `turn_speed` | `80` | 转向速度（0-255） |

### bluetooth_sender_node

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `feedback_rate` | `2.0` | 反馈发送频率（Hz） |

## 命令类型定义

| 命令 | 值 | 说明 |
|------|-----|------|
| `CMD_MODE_AUTO` | `0x10` (16) | 切换自动模式 |
| `CMD_MODE_MANUAL` | `0x11` (17) | 切换手动模式 |
| `CMD_FORWARD` | `0x20` (32) | 前进 |
| `CMD_BACKWARD` | `0x21` (33) | 后退 |
| `CMD_BRAKE` | `0x22` (34) | 制动 |
| `CMD_STOP` | `0x23` (35) | 紧急停止 |
| `CMD_LEFT` | `0x24` (36) | 左转 |
| `CMD_RIGHT` | `0x25` (37) | 右转 |

## 故障排查

### 问题：蓝牙硬件未找到

```bash
# 检查蓝牙适配器
hciconfig

# 启用蓝牙
sudo hciconfig hci0 up
```

### 问题：节点启动失败

```bash
# 重新构建
colcon build --packages-select PipeBot_driver PipeBot_bringup

# 源环境
source install/setup.bash

# 查看详细错误
ros2 run PipeBot_driver bluetooth_receiver_node --ros-args --log-level DEBUG
```

### 问题：手机扫描不到设备

1. 检查蓝牙硬件是否启用
2. 检查 bluetooth_receiver_node 是否运行
3. 检查蓝牙设备名称是否正确

### 问题：收不到命令

1. 检查所有节点是否运行
2. 检查话题是否存在：`ros2 topic list`
3. 通过 ROS2 话题模拟发送命令
