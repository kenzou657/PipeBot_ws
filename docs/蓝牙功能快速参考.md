# PipeBot 蓝牙功能快速参考卡

## 核心架构一页纸

```
┌─────────────────────────────────────────────────────────────┐
│  手机 (nRF Connect)                                         │
│  ├─ 发送命令 → 接收特征值 (UUID: ...ABCDEF1)              │
│  └─ 接收反馈 ← 发送特征值 (UUID: ...ABCDEF2)              │
└─────────────────────────────────────────────────────────────┘
                          ↕ BLE 无线
┌─────────────────────────────────────────────────────────────┐
│  树莓派 (bluetooth_receiver_node)                           │
│  ├─ 创建虚拟 BLE 设备                                       │
│  ├─ 监听手机写入 → 发布到 /bluetooth/command              │
│  └─ 订阅 /bluetooth/feedback → 发送通知给手机             │
└─────────────────────────────────────────────────────────────┘
                          ↕ ROS2 话题
┌─────────────────────────────────────────────────────────────┐
│  其他节点 (command_handler, state_manager 等)              │
│  ├─ 订阅 /bluetooth/command 处理命令                       │
│  └─ 发布到 /bluetooth/feedback 发送状态                    │
└─────────────────────────────────────────────────────────────┘
```

## 关键概念速查表

| 概念 | 解释 | 例子 |
|------|------|------|
| **Service** | 一组相关功能的集合 | `12345678-1234-5678-1234-56789ABCDEF0` |
| **Characteristic** | Service 内的具体数据 | 接收特征值、发送特征值 |
| **UUID** | 全局唯一标识符 | `12345678-1234-5678-1234-56789ABCDEF1` |
| **Notify** | 主动推送数据给客户端 | 树莓派 → 手机 |
| **Write** | 客户端写入数据 | 手机 → 树莓派 |
| **Publisher** | ROS2 发送者 | 发布命令到话题 |
| **Subscription** | ROS2 接收者 | 订阅反馈话题 |
| **GLib MainLoop** | 事件处理循环 | 处理 D-Bus 消息 |

## 代码片段速查

### 创建发布者
```python
self.pub = self.create_publisher(UInt8MultiArray, 'topic_name', 10)
self.pub.publish(msg)
```

### 创建订阅者
```python
self.sub = self.create_subscription(
    UInt8MultiArray, 'topic_name', self.callback, 10
)
def callback(self, msg):
    data = list(msg.data)
```

### 创建 BLE 外设
```python
from bluezero import peripheral
p = peripheral.Peripheral(
    adapter_address='88:A2:9E:4B:BC:E6',
    local_name='PipeBot Control'
)
```

### 添加服务
```python
p.add_service(srv_id=1, uuid='...', primary=True)
```

### 添加特征值
```python
p.add_characteristic(
    srv_id=1, chr_id=2, uuid='...',
    value=b'', notifying=False,
    flags=['write', 'write-without-response'],
    write_callback=callback_func
)
```

### 发送通知
```python
char.set_value(data_list)
char.PropertiesChanged(
    'org.bluez.GattCharacteristic1',
    {'Value': data_list}, []
)
```

### 运行 GLib 主循环
```python
from gi.repository import GLib
mainloop = GLib.MainLoop()
GLib.idle_add(function_name)
mainloop.run()
```

## 错误排查速查表

| 症状 | 可能原因 | 检查项 |
|------|--------|--------|
| 手机找不到设备 | 广播未启动 | 检查 `_start_advertising()` 日志 |
| 无法连接 | 蓝牙适配器地址错误 | 运行 `hcitool dev` 确认 |
| 接收不到命令 | 回调未触发 | 检查 GLib 主循环是否运行 |
| 发送不了数据 | 特征值引用为空 | 检查特征值是否正确保存 |
| 权限错误 | 需要 root 权限 | 使用 `sudo` 运行 |
