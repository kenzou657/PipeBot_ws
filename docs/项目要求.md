# 多功能管道内检测小车与系统

## 项目说明
针对小口径的管道，设计一个具有多种检测功能模块的小车，用于管道内的堵塞、形变等检测，该系统具备如下功能：
- 音、视频信号采集；小车与外界通信传输
- 小车管道内定位
- 基于图像的管道堵塞、形变、裂纹等缺陷检测（使用云端模型）
- 数据中台、数据云端分析

## 内容与要求
1. 多功能管道内检测小车制作：基于树莓派和ROS的车辆实物设计与制作（车辆本体购买组件），包含小车控制、通信、遥控等
2. 多功能管道内检测小车定位：基于车载陀螺仪数据的小车定位算法
3. 管道内检测图像分析：基于图像的管道堵塞/形变/裂纹等缺陷检测算法
4. 多功能管道内检测小车数据中台：展示小车摄像头数据、行程、检测结果分析与报警等
